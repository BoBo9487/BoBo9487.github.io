如何在windows下配置sqllibs

发表于 2018-03-06

今天觉得自己注入好差啊。。。所以想着来配置一下sqllibs练练手，也进了不少坑，最后还是成功了~
首先配置sqllibs需要有一个集成的mysql+php+apache环境，推荐用phpstudy和wamp（我用的）。比较方便。。。。
这里先提供sqllibs源码下载地址[链接]: https://github.com/Audi-1/sqli-labs
下载完以后先别急着解压，先配置wamp
如图，一定要保证wamp是绿色在线的，很多下载下来以后没自动开启服务，记得右键单击wamp图标重启所有服务
注意：wamp安装路径不能有中文，否则不会成功，要是遇见别的问题随时百度
把下载好的sqllibs解压到wamp安装目录里面的www这个目录里
现在配置一下mysql的密码
找到解压好的sqllibs里面的sql-connections这个文件夹，打开以后找到db-creds.inc用记事本或者sublime Text打开，如图修改一下密码，用户默认root。

配置好以后我们打开http://localhost会出现如下页面

然后我们不要直接点下面的myproject，而是输入http://localhost/sqli-labs-master/(这个坑踩了好久hhh~因为好多浏览器默认点开myproject会是新的网页而不是localhost下的子页面)
打开以后我们点击如图链接进行数据库设置

这里查到USB流量分为键盘流量和鼠标流量。
键盘数据包的数据长度为8个字节，击键信息集中在第3个字节，每次key stroke都会产生一个keyboard event usb packet。
鼠标数据包的数据长度为4个字节，第一个字节代表按键，当取0x00时，代表没有按键、为0x01时，代表按左键，为0x02时，代表当前按键为右键。第二个字节可以看成是一个signed byte类型，其最高位为符号位，当这个值为正时，代表鼠标水平右移多少像素，为负时，代表水平左移多少像素。第三个字节与第二字节类似，代表垂直上下移动的偏移。这里数据包长度是八个字节显然为键盘数据包
